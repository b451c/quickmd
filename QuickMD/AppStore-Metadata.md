## App Store Description (English)

```
QuickMD is the fastest, most elegant Markdown viewer for macOS. Double-click any .md file and instantly see beautifully rendered content.

INSTANT LAUNCH
• Opens in milliseconds — no loading screens
• Native macOS app — no Electron bloat
• Lightweight (~2MB) — respects your disk space

COMPLETE MARKDOWN SUPPORT
• Headers, bold, italic, strikethrough
• Tables with proper column alignment
• Code blocks with syntax highlighting
• Task lists with checkboxes
• Nested lists (ordered and unordered)
• Nested blockquotes with level indicators
• Images (local and remote URLs)
• Links (inline, reference-style, autolinks)
• Horizontal rules

NAVIGATION & SEARCH
• Find in document (⌘F) with per-word match count and navigation
• Word-level highlighting across all block types (code, tables, blockquotes)
• Table of Contents sidebar (⌘⇧T) — auto-generated from headings
• Copy entire document (⌘⇧C) or individual sections (hover heading → copy icon)
• Export to PDF (⌘⇧E) and Print (⌘P)

7 COLOR THEMES
• Auto (follows system), Solarized Light, Solarized Dark, Dracula, GitHub, Gruvbox Dark, Nord
• Select in Settings (⌘,) — persists across restarts

DEVELOPER-FRIENDLY
• Syntax highlighting for 10+ languages
• Perfect for README files and documentation
• Handles AI-generated markdown perfectly
• Dark mode or choose a fixed theme

PRIVACY FOCUSED
• No analytics, no tracking
• Works completely offline
• Your files stay on your device

Perfect for developers, writers, students, and anyone who works with Markdown daily. QuickMD is the Preview.app equivalent for Markdown files.
```

## Next Steps

1. **Open Xcode:**
   ```
   open "/Volumes/@Basic/Projekty/markdown-reader-MacOS/QuickMD/QuickMD.xcodeproj"
   ```

2. **Configure Signing:**
   - Go to: QuickMD target → Signing & Capabilities
   - Select your Team
   - Ensure "Automatically manage signing" is checked

3. **Archive:**
   - Product → Archive
   - Wait for build to complete

4. **Upload:**
   - Window → Organizer
   - Select archive → Distribute App
   - Choose "App Store Connect"
   - Upload

5. **App Store Connect:**
   - Go to: https://appstoreconnect.apple.com
   - Create new app with Bundle ID: pl.falami.studio.QuickMD
   - Fill in metadata from this document
   - Add screenshots
   - Submit for review

## Review Notes (optional, for Apple reviewer)

```
QuickMD is a Markdown file viewer. To test:
1. Open any .md file from Finder
2. The app will display rendered Markdown content
3. Press ⌘F to search — matches highlight in all block types, arrows navigate per word
4. Press ⌘⇧T to open the Table of Contents sidebar
5. Press ⌘⇧C to copy the entire raw markdown to clipboard
6. Hover any heading to see a copy icon — click to copy that section
7. Press ⌘, to open Settings and switch between 7 color themes
8. Test dark mode by switching system appearance
9. Press ⌘⇧E to export to PDF

No login required. No network access needed for core functionality.
```
